Nossa Abordagem

Criamos aplicativos simples e objetivos, pensados para resolver problemas reais do dia a dia. Cada projeto nasce como uma ferramenta prática para facilitar tarefas específicas — seja lembrar medicamentos no momento certo ou ajudar alguém a desacelerar em momentos de ansiedade.

Remédio Certo

🔒 PRIVACIDADE OFFLINE · SQLITE

Assistente pessoal para controle de medicamentos, desenvolvido especialmente para idosos, cuidadores e familiares. O grande diferencial? Alarme com foto do medicamento — quando chega a hora, o app exibe a imagem do remédio cadastrado, evitando confusões em tratamentos com múltiplas medicações.

  • ✅ Cadastro completo de medicamentos com fotos
  • ✅ Alertas automáticos com identificação visual
  • ✅ Histórico detalhado de doses tomadas
  • ✅ Interface simples e acessível
  • ✅ 100% offline — dados armazenados localmente com SQLite

"Não apenas lembra de tomar, mas mostra exatamente qual remédio tomar."

Respira App

☁️ CLOUD SYNC · FIREBASE

Suporte emocional para momentos de ansiedade. O Respira combina cronômetro para crises com técnicas guiadas de respiração e registro de estado emocional. Após cada crise, o usuário registra como se sente — e o app transforma esses dados em gráficos de evolução emocional para acompanhamento.

  • ✅ Cronômetro específico para crises de ansiedade
  • ✅ Técnicas guiadas de respiração
  • ✅ Registro pós-crise com análise emocional
  • ✅ Histórico e gráficos de evolução
  • ✅ Sons relaxantes e notificações
  • ✅ Login com Google/Firebase
  • ✅ Plano Premium + pacotes extras de sons

"Seus dados sempre com você."

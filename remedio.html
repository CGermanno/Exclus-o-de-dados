Ao baixar e utilizar o aplicativo Remédio Certo, você concorda expressamente com os termos abaixo.

O Remédio Certo é uma ferramenta de apoio. O usuário é o único responsável por tomar sua medicação. Falhas tecnológicas acontecem. Se o alarme não tocar, a responsabilidade pela ingestão do medicamento continua sendo do paciente/cuidador.

1.1. Permissões e Funcionamento Técnico

Para que o aplicativo funcione, é obrigatório que o usuário conceda as permissões de:

  • Alarmes e Lembretes (Schedule Exact Alarm);
  • Notificações;
  • Sobreposição de Tela (opcional, mas recomendado).

Atenção: Se o usuário negar essas permissões, ativar o modo "Não Perturbe", "Economia de Bateria" ou utilizar aplicativos de limpeza de memória (Task Killers), o alarme poderá falhar. O desenvolvedor não se responsabiliza por omissões de doses causadas por configurações restritivas do aparelho do usuário.

1.2. Otimização de Bateria (Samsung, Xiaomi, etc.)

Alguns fabricantes de celular "matam" aplicativos em segundo plano para economizar bateria. É dever do usuário garantir que o Remédio Certo esteja na lista de exceções de economia de bateria do seu aparelho para garantir o funcionamento do despertador.

2. Política de Privacidade

Em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018) e as diretrizes de segurança da Google Play, detalhamos como seus dados são tratados.

2.1. Modelo de Armazenamento (Offline-First)

O Remédio Certo adota uma política de Privacidade por Design. Diferente de outros aplicativos, nós NÃO possuímos servidores, bancos de dados em nuvem ou sistemas de login.

Todos os dados inseridos (nomes de medicamentos, fotos, horários, histórico de consumo) são gravados exclusivamente no banco de dados local (SQLite) dentro do seu próprio dispositivo. Nós, desenvolvedores, não temos acesso técnico a essas informações.

2.2. Uso de Permissões do Android

Para o funcionamento correto, o aplicativo solicita as seguintes permissões estritamente necessárias:

  • Notificações (POST_NOTIFICATIONS): Necessária para exibir os alertas de hora de tomar o remédio.
  • Alarmes Exatos (SCHEDULE_EXACT_ALARM): Necessária para garantir que o alarme toque no minuto exato, mesmo se o celular estiver em repouso.
  • Sobreposição de Tela (SYSTEM_ALERT_WINDOW): Opcional. Permite que o alarme apareça sobre outros apps quando tocar.
  • Câmera e Armazenamento: Utilizadas apenas se o usuário optar por tirar foto da caixa do remédio. A imagem permanece salva apenas localmente.

2.3. Coleta de Dados Pessoais

O aplicativo não coleta:

  • Dados de localização (GPS);
  • Identificadores de publicidade (Advertising ID);
  • Dados de saúde para envio externo.

3. Exclusão de Dados

Como não mantemos contas de usuário ou servidores centrais, o usuário possui total soberania sobre seus dados.

Para excluir definitivamente suas informações:

  1. Método Simples: Desinstale o aplicativo do seu celular. O sistema operacional Android excluirá automaticamente todo o banco de dados e arquivos associados.
  2. Método Manual: Acesse Configurações > Aplicativos > Remédio Certo > Armazenamento > Limpar Dados.

Não é necessário enviar e-mail solicitando a exclusão, pois tecnicamente não possuímos acesso aos seus dados para apagá-los remotamente.
